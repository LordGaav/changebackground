Random Wallpaper Changer for Compiz
-----------------------------------

This is a script to randomly change your wallpaper. It uses a folder of images as input, and can handle multiple workspaces and multiple monitors.

Dependencies
------------

* php5-cli
* php5-imagick
* graphicsmagick
* imagemagick
* compiz (< 0.9.8 or >= 0.9.9)
